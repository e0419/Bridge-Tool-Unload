# Bridge-Tool-Unload
Unified program for unloading of tools from 3-axis bridge mills.
 - Main branch configured for Takumi H12e program.
 - H16 Branch uses same code, different tool counts and machine offsets
 - Kitamura Bridge 6g branch also includes updated variables for Mitsubishi based Arumatik conroller.
 - All branches set to clear tool geometry and wear offsets after each tool is unloaded.
